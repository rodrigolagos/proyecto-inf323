# proyecto-inf323

### Requerimientos
* Node JS
* npm
* MongoDB

### Iniciar el proyecto
Clonar el repositorio

`git clone https://github.com/rodrigolagos/proyecto-inf323.git`

Iniciar mongoDB (en una terminal aparte)

Unix: `sudo mongod`

Windows: `mongod`

Iniciar el proyecto

`npm install`

`npm start`
